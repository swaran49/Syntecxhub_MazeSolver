# Maze Solver using A* Search

This project implements the A* Search Algorithm to solve a maze and find the shortest path.

## Features
- A* Search Algorithm
- Animated pathfinding
- Moving AI agent
- Random maze generation
- Execution time calculation
- Visited nodes counter
- Visual maze interface

## Technologies Used
- Python
- pyamaze

## How to Run

Install dependency:

pip install pyamaze

Run project:

python main.py

to learn more about A* Search:

 https://www.geeksforgeeks.org/dsa/a-search-algorithm/
